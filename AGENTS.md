# Project Navigation and Terminology Rules

To provide persistent guidance across different sessions, please adhere strictly to the following terminology and visual mapping for the navigation elements in this project:

1. **Ecosystem (First Tab / Dashboard):** The first section where the user's active learning packages are displayed and managed.
2. **Discovery (Second Tab / Discover):** The section where new packages are available to be searched, explored, and added.
3. **Forum (Third Tab / Salotto):** The community space, also referred to as "Il Salotto".
4. **Dictionary (Fourth Tab / Dictionary):** The interactive lexicon and vocabulary reference.
5. **Account (Fifth Section / Profile):** Relocated directly to the Header Profile Avatar (the 'Salman Arab' trigger area), which reveals the user profile and stats drawer.

Always refer to, structure, and name these views according to this taxonomy. Do not modify these architectural boundaries or terms.
